About the Project

MealBridge is a platform where Donors can list surplus meals, and Collectors (NGOs) can accept and distribute them to those in need.

This backend powers authentication, user management, donation posting, collection handling, real-time updates via Socket.IO, and secure communication between donors and collectors.

API Overview

Route Description
/authentication Login, signup, and session handling using Passport.js
/user Universal user profile endpoints
/donor Endpoints for donors to post meals & manage profiles
/collector Endpoints for collectors (NGOs) to view & accept donations
/meal Handles OTP verification for meal collection
